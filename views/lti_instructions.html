ACOS is an open-source advanced content server designed to provide online learning activities to be used in different learning management systems that use different protocols including LTI. It contains multiple content types and learning materials. The current list of available content to be used in Canvas can be checked here. 

To add ACOS exercises in your Canvas course, please follow the steps below:

  1. Given that the ACOS server is meant to be light-weight and just serving content, currently, it does not support account management services. To obtain a consumer key and secret by sending an email to kaa108@pitt.edu.
  2. Add ACOS server as an external application.

Step 1:  Navigate to your course in Canvas and click the "Settings" link from the left sidebar.

Step 2: Go to "Apps" and click on "View App Configurations". Note that your institution's version of Canvas might display differently.

Step 3: Click on the "+App" button. 

Step 4: From configuration type, select "By URL" option. 

Step 5: Paste your consumer key and the shared secret that you obtained in Step 1. After that, paste https://acos.cs.vt.edu/lti/lticonfig.xml under "Config URL". This is the URL to the XML configuration for ACOS Server.

Step 6: After you hit submit, you will be able to see ACOS Server as an external tool option while creating your graded or ungraded learning activities.
